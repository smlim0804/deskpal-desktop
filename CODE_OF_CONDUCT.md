# Code of Conduct

DeskPal uses a simple rule: be clear, kind, and practical.

Contributors and users are expected to:

- Treat others with respect.
- Keep feedback focused on the work.
- Avoid harassment, personal attacks, or discriminatory language.
- Report security issues privately instead of posting exploit details in public issues.

Maintainers may edit, hide, or remove comments and contributions that make the project unsafe or hostile for others.
